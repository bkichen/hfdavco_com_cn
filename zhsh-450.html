非沥青基高分子自粘防水卷材

产品概述   ▄PRODUCT OVERVIEW

　　非沥青基高分子自粘防水卷材是防水工程设计广泛应用的一种新型防水材料 ; 卷材以高分子材料为基材、涂上非沥青基的环保高分子自粘胶层复合而成 ; 环保高分子自粘胶层耐老化性能大大提高，比一般高分子卷材层的老化性能更为优越。



产品特点   ▄PRODUCT CHARACTERISTICS

★非沥青基高分子自粘防水卷材兼有高分子防水卷材和自粘卷材的优点，大大提高了抗穿刺、自愈、耐高、低温等性能，化学性能更稳定 ;

★非沥青基高分子自粘防水卷材采用自粘施工工艺，施工更为方便 ;

★自粘胶层采用本公司研制的环保高分子自粘胶，耐老化性能大大提高，保证了自粘效果的性 ;

★非沥青基高分子自粘防水卷材实现了高分子卷材不窜水的可能 ;

★安全环保 ; 高分子基材与自粘胶料，均采用无害、无毒的环保配方 。



适用范围   ▄CONSTRUCTION MRTHOD

★屋面、地下室、地下车库、人防等建筑工程 ;

★水库、堤坝、围堰、渠道等水利工程 ;

★桥梁、地铁、公路、涵洞等交通工程 ;

★屋顶花园、种植屋面、园林等绿化工程。
